# Plan 1 – Hero section & visual identity

## Prompt a la IA
"Help me design an organic animated hero background inspired by Oversour Studio using CSS gradients and Bootstrap."

## Respuesta de la IA (resumen)
- Usar gradientes animados con background-position
- Tipografía editorial con contraste
- Mantener accesibilidad y prefers-reduced-motion

## Implementación
- Se implementó hero con gradiente animado
- Tipografía Instrument Serif + DM Sans
- Animación desactivable con prefers-reduced-motion






## Use of AI tools

This project used AI assistants during planning phases only.
Each AI interaction followed a two-step process:
1. Planning (documented in /docs)
2. Manual implementation and adaptation
